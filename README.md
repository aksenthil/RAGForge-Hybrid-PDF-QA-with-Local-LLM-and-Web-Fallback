# RAGForge-Hybrid-PDF-QA-with-Local-LLM-and-Web-Fallback
Local-first RAG app: upload PDFs, hybrid chunking, hybrid retrieval (dense + TF‑IDF) with RRF + MMR/Cross‑Encoder rerank. Grounded, cited answers via GPT4All/llama.cpp. Diagnostics and optional web fallback. Streamlit UI, disk cache. Windows/macOS/Linux.
